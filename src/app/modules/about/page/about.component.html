À propos de l'application

Le but de cette application est de mettre à disposition des utilisateurs un dashboard permettant de visualiser les informations des précédents Jeux olympiques (nombre de médailles par pays, etc.).

Elle est aussi une base d'étude d'Angular:

  • Architecture modulaires
  • Composants
  • Liaison de données (interpolation, liaison de données etc.)
  • Services
  • Utilisation des observables
  • Directives
  • Routing

Elle repose sur la librairie de graphiques de CanvasJS.
